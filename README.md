## Project Overview
This is a **printer job scheduling system** implemented in **C++**, designed to efficiently manage and process print jobs based on **priority**. It demonstrates practical usage of **data structures and algorithms**, specifically **circular linked lists** and **priority queue logic**.

## Features
- **Circular Linked List** for dynamic and efficient queue management.
- **Priority-based insertion**: higher-priority jobs are processed first.
- Add, delete, search, and view jobs in the queue.
- Track **total jobs processed** and **total pages printed**.
- Persistent storage using **file I/O** (`queue.txt`) to save and load queue data.
- Displays processing time for each operation using **microsecond-level timing**.
- Robust input validation and error handling.

- Dynamic memory management (pointers)
- Algorithmic complexity optimization
